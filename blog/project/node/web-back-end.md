## HTTP 协议

### 是什么?

- HTTP是一个客户端（用户）和服务端（网站）之间请求和应答的**标准**，通常使用[TCP协议](https://zh.wikipedia.org/wiki/传输控制协议)。

- 分为请求协议与响应协议



### 作用是什么? + 如何实现目标?

- 传输数据

  核心目的是传输数据, 在请求/响应时, 提前对数据的类型, 格式进行说明, 便于高效处理数据.

  `curl -v https://www.baidu.com`
