### 【新知识】如何做错误处理？

- 分为业务错误与非业务错误

  - 业务错误，httpcode === 200，具体业务错误数据结构如下

    ```
    {
    	code: '00000',// 业务结果,00000表示业务正常
    	data: any,
    	result: 'ok' // 这里返回业务错误信息
    }
    ```

  - 非业务错误，httcode !==200 ,会被前端axios封装层提前处理，具体数据结构如下

    ```
    {
        "statusCode": 400,
        "message": "email is invalid",
        "error": "Bad Request"
    }
    ```
