## 完全居中

1. 绝对定位法

   - 前提：①父div和子div元素的height和width都已知

   - 方法：

     1. 给父元素加`position: relative`
     2. 给居中元素加

     ```
     position: absolute;
     top: 0;
     left:0;
     right: 0;
     bottom: 0;
     margin: auto;
     ```

     ​

2. flex定位：

   - 前提：有父div和子div，父div和子div元素的height和width都已知，但子div的position已经确定了

   - 方法：

     给父div

     ```
     display: flex;justify-content: center;align-items:center;
     ```

     ​
