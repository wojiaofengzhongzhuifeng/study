## 函数定义

### 方式

- 函数声明

  ```javascript
  function test(){}
  ```

- 函数表达式

  ```javascript
  var test = function(){}
  ```

- 区别
  1. 函数声明必须有函数名称；函数表达式可以是匿名函数，并 把这个函数赋值给变量。
  2. 函数声明会提升；函数表达式不会。

### 使用场景

- 斐波那锲数列（函数定义处调用自身）

  ```javascript
  function fibonacci(number){
  	if(number === 1 || number === 2){
  		return 1
  	}
  	return fibonacci(number -1) + fibonacci(number -2)
  }
  ```

- 模块化

  想实现的通过 var 定义的变量是块级作用域功能

  ```javascript
  var getMoneyObj = function(){
  	var money = 1
  	return {
  		getMoney: function(){
  			console.log(money)
  		}
  	}
  }()
  getMoneyObj.getMoney()
  ```
